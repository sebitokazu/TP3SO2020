# TP3SO2020: Sockets

## Instrucciones de compilacion

Run the following command in the project directory:

```
user@linux:$ make all
```

## Instrucciones de uso

Para correr el server:

```
user@linux:$ ./server
```

Para correr el cliente:

```
user@linux:$ ./client ip port
```
